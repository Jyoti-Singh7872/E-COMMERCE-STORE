
# E-Commerce Store Data Analysis

## Overview
This project focuses on analyzing the E-Commerce Store dataset to extract valuable insights related to customer behavior, sales performance, and trends. Python libraries like pandas, matplotlib, and seaborn were utilized for data cleaning, exploration, and visualization.

## Objectives
- Clean and preprocess the dataset.
- Analyze customer purchasing behavior and sales trends.
- Identify top-selling products and profitable regions.
- Provide actionable insights for improving business strategies.

## Dataset

The dataset contains 541,909 records and 8 features:
- `InvoiceNo`: Unique identifier for each transaction.
- `StockCode`: Unique product identifier.
- `Description`: Description of the product.
- `Quantity`: Number of items purchased.
- `InvoiceDate`: Date of the transaction.
- `UnitPrice`: Price per item.
- `CustomerID`: Unique identifier for each customer (may have missing values).
- `Country`: Country where the transaction occurred.

## Key Steps
1. **Data Cleaning**:
   - Checked for missing values and handled them appropriately.
   - Dropped rows with missing `CustomerID`.
   - Converted columns to suitable data 

types.
2. **Exploratory Data Analysis (EDA)**:
   - Analyzed sales trends and seasonal patterns.
   - Examined customer segmentation.
   - Visualized the top-selling products and countries.
3. **Visualization**:
   - Used matplotlib and seaborn to create graphs and charts.
   - Highlighted insights in a meaningful way.
4. **Insights and Recommendations**:
   - Shared actionable insights to improve business operations.
